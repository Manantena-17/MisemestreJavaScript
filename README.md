# 💰 FinanceFlow - Personal Finance Management Web App

FinanceFlow is a modern web application designed to help users take full control of their financial life. It allows you to track budgets, manage transactions, and visualize your financial health through an intuitive dashboard.

## 🚀 Key Features (Current)

* **Financial Dashboard**: A comprehensive overview of your total balance, income, and expenses with visual charts.
* **Budget Management**: Set spending limits for different categories.
* **Transaction Tracking**: Detailed history of all your financial movements with easy filtering.
* **Customizable Settings**: Personalize your dashboard display and currency preferences.

## 🛠️ Tech Stack

* **Language**: JavaScript
* **Styling**: CSS
* **Deployment**: Hosted on **Vercel** 🚀
